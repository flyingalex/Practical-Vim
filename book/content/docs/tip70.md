### 70. Evaluate an Iterator to Number Items in a List

An example:
```
:let i=1
qa
I<C-r>=i<CR>)<Esc>
:let i += 1
q
```
After you record the macro,you can use VISUAL MODE to select all lines you want to act this and call it.
