### 77. Stake the Boundaries of a Match

`\zs` and `\ze` can handle this stuff.These two command make just highlight the words you want.`\zs` for start and `\ze` for end.

An example:
```
Practical Vim
```

Command `/Practical \zsVim<CR>` will highlight the `Vim`.
